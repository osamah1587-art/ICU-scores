# ICU Scores

30 critical care scoring systems in a single PWA — designed for bedside use.

## Scores included
APACHE II, BISAP, CAM-ICU, Candida Score, CHA₂DS₂-VASc, CPIS, CURB-65, DIC (ISTH), FOUR Score, GCS, HACOR, HAS-BLED, HEART Score, KDIGO AKI, mBISAP, NIHSS, PESI/sPESI, Ranson, RIFLE, SAPS II, SOFA, STOP-BANG, SAVE/STS-ECMO, Wells DVT, Wells PE

## Features
- Full clinical decision support per score (result + recommendation + notes + reference)
- Search + favourites
- Dark mode, PWA, offline support
- Install to home screen (no browser bar)

## Deploy
1. Create GitHub repo `icu-scores`
2. Upload all files
3. Settings → Pages → main / root
4. Live at `https://YOUR-USERNAME.github.io/icu-scores/`
